Welcome to PLASTECURE

PLASTECURE is an innovative solution for managing plastic waste efficiently and responsibly. Our mission is to create an eco-friendly system that tracks, recycles, and ensures sustainable plastic management.

Features

  • IoT Integration: Monitors and tracks recycling activities in real-time using ESP32.
  • Supabase Database: Stores and manages recycling data securely.
  • User-Friendly Dashboard: Visualizes data and provides insights into recycling efforts.
  • Automation: Automatically logs recycling activity and tracks progress.
  • Eco-Friendly Focus: Promotes responsible plastic disposal and recycling practices.

How to Set Up

  1. Assemble the hardware with ESP32 and sensors.
  2. Install the required software and flash the firmware.
  3. Connect the device to Supabase for real-time updates.
  4. Access the dashboard for monitoring and analytics.

FAQs

What type of plastic can I recycle?

PLASTECURE supports all commonly recyclable plastics. Refer to the dashboard for specifics.

Is PLASTECURE offline-compatible?

Currently, an internet connection is required for full functionality.
